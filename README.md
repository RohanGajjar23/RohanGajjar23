  - Shipped across Android, iOS, and WebGL with login paths and cloud-backed sync

  [App Store](https://apps.apple.com/us/app/imposter-party-multiplayer/id6756916475)

  - Added premium subscription flows, analytics, and release-ready support for Android, iOS, and Windows

  - Built room systems, pet interactions, and modular event-driven gameplay
  - Implemented 8+ mini-games on shared systems for faster content iteration
  - Worked on Supabase integration, secure APIs, wallet features, and cross-platform optimization

- **Rohan's Tax Services** -- a production Flutter app that proves I can also ship secure, connected apps outside the game space when a product needs it\
  [Google Play](https://play.google.com/store/apps/details?id=com.rgstudios.rohanstaxservicescloud)
  - Built and launched a client-facing app for secure document access, reminders, and updates
  - Implemented .NET 9 APIs on AWS with JWT auth, encrypted S3 storage, and DynamoDB workflows
  - Released across web, Google Play, and the Apple App Store
