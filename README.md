## About Me

I am a game developer based in Ahmedabad, India, with hands-on experience building gameplay systems, multiplayer flows, monetization features, and release-ready products across WebGL, mobile, and desktop targets.

My strongest lane is Unity and C#, especially when a project needs smooth player experience, reliable multiplayer behavior, and the final polish that gets a build from "working" to "ready to ship." Alongside games, I have also shipped production app work with Flutter, .NET, AWS, and Supabase, which helps me think beyond isolated gameplay code and build connected products end to end.
